# Echo theme for Bludit

Bludit and Bootstrap version of original Echo theme for Nibbleblog by Diego Najar.

## Compatibility

- Branch `master` — Bludit v3.5.x and up
- Branch `2.3` — Bludit v2.0.x – v2.3.x

## Versioning

This theme follows a **mirror versioning** scheme tied to Bludit CMS releases:

- **Major.Minor** mirrors the compatible Bludit stable release (e.g. `3.5.x` targets Bludit 3.5).
- **Patch** increments independently with each theme release.

## Releases

Releases are created automatically via GitHub Actions when a version tag is pushed:

```
git tag 3.5.1
git push origin 3.5.1
```
